# Step 7: Workflow - Initial Setup

You have been assigned an issue. Now it's time to get the code onto your computer.

## 🍴 1. Fork the Repository
Go to the project's GitHub page and click the **"Fork"** button. This creates a personal copy of the project in your own GitHub account.

## 👥 2. Clone Your Fork
Copy the URL of *your* fork and type this in your terminal:
```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

## 🌿 3. Create a Branch
Create a branch specifically for the fix you are making:
```bash
git checkout -b fix/typo-in-guide
```
