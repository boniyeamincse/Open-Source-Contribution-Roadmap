# Step 3: Setting Up Your Tools

To collaborate on code, you need a few industry-standard tools. Don't worry—most are free!

## 💻 1. Pick a Code Editor
You need a place to write code. 
- **Recommendation**: [Visual Studio Code (VS Code)](https://code.visualstudio.com/) is the most popular and has great Git integration.

## 🌲 2. Install Git
Git is the system that tracks your changes.
- **Windows/Mac/Linux**: [Download Git here](https://git-scm.com/downloads).
- **Verify**: Type `git --version` in your terminal to check if it's installed.

## 🌐 3. Create a GitHub Account
GitHub is where the open-source world lives.
- Sign up at [GitHub.com](https://github.com/).

## ⚙️ 4. Essential Configuration
Tell Git who you are (this will show up in your contributions):
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```
