# Learning Objectives

This document outlines the specific skills and knowledge you will gain as you journey through this roadmap.

## 🟢 [Module 1: Introduction to Open Source](./Module-01-Introduction-to-Open-Source/README.md)
- Define what Open Source is and its core principles.
- Understand the difference between Proprietary and Open Source software.
- Identify the benefits of contributing (Experience, Network, Portfolio).
- Learn about the OSS ecosystem: Maintainers, Contributors, and Users.

## 🔵 [Module 2: Git and Version Control](./Module-02-Git-and-Version-Control/README.md)
- Explain the concept of Version Control Systems (VCS).
- Master basic Git commands: `git init`, `git add`, `git commit`, `git status`.
- Understand Branching and Merging strategies.
- Interface with GitHub: Forking, Cloning, and Pushing code.

## 🔴 [Module 3: Understanding Open Source Projects](./Module-03-Understanding-Open-Source-Projects/README.md)
- Read and interpret standard project files (`README`, `LICENSE`, `CONTRIBUTING`).
- Understand Issue Trackers and how to find "Good First Issues".
- Learn the etiquette of communication in open-source communities.
- Understand basic Open Source licenses (MIT, Apache, GPL).

## 🟡 [Module 4: First Contribution](./Module-04-First-Contribution/README.md)
- Follow the end-to-end contribution workflow.
- Create descriptive commits and pull request descriptions.
- Respond to code review feedback.
- Celebrate your first merged contribution!
