# Module 2 Self-Test: Git and Version Control

Verify your technical understanding of Git.

## ❓ Questions

1.  **Which command initializes a new Git repository?**
    - A) `git start`
    - B) `git status`
    - C) `git init`
    - D) `git new`

2.  **What does "Staging" a change mean in Git?**
    - A) Saving it to the history forever.
    - B) Telling Git to include it in the next commit.
    - C) Deleting the file.
    - D) Sending it to GitHub.

3.  **Which command is used to record changes to the local history?**
    - A) `git add`
    - B) `git push`
    - C) `git commit`
    - D) `git record`

4.  **What command allows you to see the log of past commits?**
    - A) `git show`
    - B) `git log`
    - C) `git history`
    - D) `git list`

5.  **Which command switches to a new branch?**
    - A) `git checkout -b branch-name`
    - B) `git switch -n branch-name` (Note: both are valid, but we used checkout in the notes)
    - C) `git branch-move`
    - D) `git go`

---

## 🔑 Answer Key
1. C
2. B
3. C
4. B
5. A
