# Module 2: Git and Version Control

In this module, you will learn the essential tools for collaboration. Git is the language of open source, and mastering it is your first step toward becoming a contributor.

## 📁 Module Contents
- [notes.md](./notes.md): Introduction to Version Control and basic Git commands.
- [examples.md](./examples.md): Common Git workflows in open-source projects.
- [lab.md](./lab.md): Hands-on project to practice your Git skills locally.
- [self-test.md](./self-test.md): Quiz to verify your technical understanding.

## 🎯 What You'll Learn
1.  What a Version Control System (VCS) is.
2.  The Git lifecycle: Add, Commit, Push.
3.  Working with branches.
4.  Connecting your local code to GitHub.
