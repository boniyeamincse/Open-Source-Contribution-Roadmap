# Step 10: Merging and Beyond

Congratulations! If you've reached this step, you are either waiting for your PR to be merged or you've already had your first "Green" contribution.

## 🤝 1. Handling Code Review
Maintainers might ask for changes. This is normal!
- Don't take it personally.
- Make the requested changes locally, commit, and push again.
- The PR will update automatically.

## 🎉 2. The Merge
Once your code is merged, it becomes part of the project forever. Celebrate! You are now an open-source contributor.

## 🚀 3. What's Next?
The journey doesn't end here.
- **Support Others**: Help new contributors in the same project.
- **Explore More**: Find bigger tasks or new repositories.
- **Share**: Tell others about your journey.

## 📚 Further Resources
- [GitHub Skills](https://skills.github.com/): Interactive courses.
- [First Timers Only](https://www.firsttimersonly.com/): A great place to find beginner-friendly work.
- [Open Source Guide](https://opensource.guide/): Comprehensive deep-dives.
