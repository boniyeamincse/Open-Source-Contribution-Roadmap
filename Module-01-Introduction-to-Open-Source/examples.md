# Module 1 Examples: Real-World Open Source

Open source isn't just hobby projects; it powers the modern world. Here are some examples of projects you probably interact with every day.

## 📱 Operating Systems
- **Linux**: The kernel that powers Android phones, supercomputers, and millions of servers.
- **Chromium**: The open-source foundation behind Google Chrome and Microsoft Edge.

## 🛠️ Developer Tools
- **VS Code**: One of the most popular code editors is open-source!
- **React/Vue/Angular**: The frameworks used to build the websites you visit.
- **Python/Go/Rust**: Major programming languages are developed in the open.

## 🌐 Global Impact
- **Apache/Nginx**: The web servers that deliver most of the web.
- **Kubernetes**: The technology that companies like Google and Netflix use to manage their apps.

## 💭 The "First Contribution" Story
Many famous developers started small:
- **Linus Torvalds** started Linux as a small side project to learn.
- **A student** might fix a typo in the VS Code documentation. That's a valid and helpful contribution!
