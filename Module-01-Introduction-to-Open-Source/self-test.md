# Module 1 Self-Test: Introduction to Open Source

Test your knowledge of the basics before moving on to Module 2.

## ❓ Questions

1.  **What is the defining characteristic of Open Source Software?**
    - A) It is always free of cost.
    - B) The source code is available for anyone to inspect, modify, and share.
    - C) It can only be used by developers.
    - D) It doesn't have any licenses.

2.  **Who is responsible for the overall direction of an open-source project and merging code changes?**
    - A) Contributors
    - B) Users
    - C) Maintainers
    - D) The Internet Service Provider

3.  **True or False: Contributing to Open Source can help you build a professional portfolio.**

4.  **What is a "Good First Issue"?**
    - A) A very complex bug that needs years of experience.
    - B) An issue labeled specifically for beginners to get started with.
    - C) An issue that no one wants to fix.
    - D) A legal problem with the project.

5.  **Which of these is a famous open-source project?**
    - A) Microsoft Word
    - B) Adobe Photoshop
    - C) Linux
    - D) iOS

---

## 🔑 Answer Key (Check yourself!)
1. B
2. C
3. True
4. B
5. C
