# Additional Learning Resources

Beyond this book, these resources will help you become a master contributor.

## 🎓 Free Courses
- **GitHub Learning Lab**: [lab.github.com](https://lab.github.com/) - Integrated Git and GitHub tutorials.
- **Intro to Open Source (Udacity)**: [udacity.com/course/software-development-process--ud805](https://www.udacity.com/course/software-development-process--ud805)
- **First Contributions**: [github.com/firstcontributions/first-contributions](https://github.com/firstcontributions/first-contributions) - A hands-on tutorial specifically for making your first PR.

## 📚 Documentation
- **Git Documentation**: [git-scm.com/doc](https://git-scm.com/doc) - The official Git "Pro" book is available here for free.
- **GitHub Docs**: [docs.github.com](https://docs.github.com/) - Comprehensive guides on all things GitHub.
- **Open Source Guide**: [opensource.guide](https://opensource.guide/) - Created by GitHub, this is an excellent high-level overview.

## 🌍 Communities
- **Up For Grabs**: [up-for-grabs.net](https://up-for-grabs.net/) - A place to find projects looking for help.
- **CodeTriage**: [codetriage.com](https://www.codetriage.com/) - Get a regular email with an issue to fix.
- **AnitaB.org / Outreachy**: Great communities focused on diversity in tech and open source.
