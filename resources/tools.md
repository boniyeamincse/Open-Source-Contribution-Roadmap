# Recommended Tools for Open Source

To be an efficient contributor, you need the right tools. Here is a curated list for beginners.

## 💻 Code Editors
- **Visual Studio Code (VS Code)**: The industry standard. Highly customizable with extensions.
- **Sublime Text**: Lightweight and fast.
- **NeoVim**: For those who love the keyboard and terminal.

## 🌲 Git Clients (GUIs)
If the terminal is intimidating, try these:
- **GitHub Desktop**: Simple and integrated with GitHub.
- **GitKraken**: Powerful and visual branching graphs.
- **Sourcetree**: A free Git client for Windows and Mac.

## 🌐 API & Documentation
- **Postman**: For testing web APIs.
- **Dash / DevDocs**: Offline documentation browsers.
- **Hoppscotch**: Open-source alternative to Postman.

## 💬 Communication
- **Discord / Slack**: Often used for community "office hours".
- **Discourse**: For long-form community discussions.
- **Matrix / IRC**: Frequently used by older or deeply technical projects (like the Linux Kernel).
